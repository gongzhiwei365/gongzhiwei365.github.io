ADB免ROOT卸载系统软件

发表于 2020-01-23 | 分类于 搞机

前言

（第一段是瞎逼逼，具体操作直接从第二段开始。）到目前为止，我还没有将IOS设备作为主力使用机器过，所以我暂时对IOS的了解不够。从初中到现在用过的安卓手机5、6部应该是有的，每一部手机我应该都ROOT过，可能没有长时期保留ROOT权限，但是绝对是进行过获取ROOT这样的一个操作。在安卓6.0以前，安卓手机用户想要自己配置软件权限是需要借助第三方工具的，并且这些应用中大部分是需要我们有超级用户（ROOT）权限的。早期的一些有名的软件，例如360手机卫士极客版、LBE安全管家，它们都是小巧而强大的手机助手软件，可以帮助我们控制软件自启动和相互唤醒、软件权限配置、手机垃圾清理、电话短信拦截等工作。后来随着安卓版本的更新优化和各家厂商内置安全中心，这些曾经名震一时的软件逐步淡出人们的视野。前几年安卓手机厂商们喜欢在自己的手机中内置一些自家的应用，并让用户无法卸载。而且越来越多的品牌考虑到用户的安全性已经不提供手机解BootLoader锁的功能（不解锁的手机无法刷机和ROOT），所以用户不能按照自己的意愿删除手机厂商内置的一些软件，即使用户并不需要这个软件。好在国家前几年出了法规要求手机厂商除了手机最基本的功能，不能将其他内置软件变为系统软件阻碍用户删除。法规有效但是还是有一部门软件是我们所不需要的，比如小米手机内置的广告服务软件，所以我们需要借助ADB工具来进行操作。

ADB工具下载

我们前去Google提供的下载网页下载（ https://developer.android.com/studio/releases/platform-tools.html），我使用的是Win版本的adb工具。

下载完成解压到你想要的目录，进入目录后，点击文件管理器的地址栏，系统会自动帮你全选当前文件夹的地址，之后我们复制此地址。使用WIn+R，调出运行，输入cmd，进行命令行控制台，输入

然后回车，此时我们就进入到了ADB工具工作的目录下。

详细操作

手机连接电脑，并打开手机USB调试。各品牌打开方式不尽相同，可自行百度。

  1. 输入 adb shell 。
  2. 输入pm list packages。查看所有的包名，找出你想卸载应用的包名，这一步你可以省略，因为很多手机可以在设置应用管理里直接看到包名，包名的结构为com.xxx.xxx，而不是软件名称。
  3. 当然你也可以先打开想卸载的软件，然后电脑输入dumpsys window | grep mCurrentFocus，就可以查看当前正在手机界面允许的软件包名。
  4. 输入pm uninstall -k --user 0 packageName。其中，-k为保留软件数据的意思，根据个人意愿可加可不加。
  5. 提示Success的话，则卸载成功。
  6. 输入exit，退出shell模式。再次输入exit，退出cmd。当然你也可以直接点击X来关闭窗口。

其他知识

第一步输入adb devices，可以查看手机是否连接电脑成功。输入这个或者第一步的时候，手机可能会弹窗提示是否允许当前电脑进行调试。还有就是在操作完成后，记得关闭手机的USB调试功能，确保手机的安全。

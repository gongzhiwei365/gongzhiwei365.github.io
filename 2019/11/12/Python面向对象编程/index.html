继承和多态

最开始的时候就说过了class类名称后的括号中一般默认写的是object，那是因为所有的类都是继承于object这个初始对象的。当我们已经有了一个对象，希望对它进行继承的话，我们只需要将括号中的内容换成父类名称就OK了。

其次，子类和父类的关系本质上是一个树状的关系，子类属于自己的单独一类，但是子类同时也是父类的一种，但是父类却并不属于子类的一种。在python中我们可以使用isinstance()这个函数去判断两个对象是不是相同种类。

这就是多态的概念，Animal类可以生出Dog类、Cat类等等，对于Animal的一些方法，它的子类都可以调用，提高了我们编程的系统性。一些对于Animal的通用操作，我们可以在Animal中直接定义好，它的子类的独特方法再单独定制。
